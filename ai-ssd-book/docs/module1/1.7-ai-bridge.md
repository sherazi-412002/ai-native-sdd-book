# Chapter 1.7: Bridging AI Agents to ROS 2

## Introduction

The integration of Artificial Intelligence (AI) agents with ROS 2 represents a revolutionary approach to robotic development. As AI capabilities continue to advance, the ability to seamlessly connect AI systems with robotic frameworks becomes increasingly critical. This chapter explores how to bridge AI agents with ROS 2, focusing on practical implementations using tools like LangChain and GPT-4o.

This integration opens up new possibilities for autonomous decision-making, natural language interaction, and intelligent robot behavior that was previously impossible with traditional programming approaches.

## The Need for AI Integration

### Traditional Robotics Limitations

Traditional robotic systems rely heavily on predefined rules and explicit programming. While effective for well-defined tasks, this approach has limitations:

- **Limited Adaptability**: Cannot easily adapt to new situations
- **Complex Rule Management**: Managing complex rule sets becomes unwieldy
- **Human Intervention Required**: Requires constant human oversight for new scenarios
- **Scalability Issues**: Scaling to more complex tasks requires extensive reprogramming

### AI-Enhanced Robotics Benefits

AI integration addresses these limitations by:

- **Adaptive Behavior**: Learning from experience and adapting to new situations
- **Natural Interaction**: Enabling human-like communication with robots
- **Intelligent Decision-Making**: Making complex decisions based on environmental data
- **Reduced Programming Overhead**: Less explicit programming for routine tasks

## AI Agent Integration Patterns

### 1. Natural Language Processing (NLP) Integration

AI agents can interpret natural language commands and translate them into robot actions:

```python
# Example: NLP to ROS 2 command translation
import openai
import rclpy
from rclpy.node import Node
from std_msgs.msg import String

class NLPRobotController(Node):
    def __init__(self):
        super().__init__('nlp_robot_controller')
        self.publisher = self.create_publisher(String, 'robot_commands', 10)

        # Initialize AI model
        self.openai_client = openai.OpenAI(api_key='your-api-key')

    def process_command(self, natural_language_text):
        # Convert natural language to structured command
        response = self.openai_client.chat.completions.create(
            model="gpt-4o",
            messages=[
                {"role": "system", "content": "Convert natural language commands to structured robot actions"},
                {"role": "user", "content": natural_language_text}
            ]
        )

        # Extract structured command
        command = response.choices[0].message.content

        # Publish command to ROS 2
        msg = String()
        msg.data = command
        self.publisher.publish(msg)

        return command
```

## LangChain Integration with ROS 2

### What is LangChain?

LangChain is a framework for developing applications powered by language models. It provides tools for chaining together language model calls, managing memory, and integrating with external tools.

### Basic LangChain Setup for ROS 2

```python
# Example: LangChain integration with ROS 2
from langchain_openai import ChatOpenAI
from langchain_core.prompts import PromptTemplate
from langchain_core.output_parsers import StrOutputParser
import rclpy
from rclpy.node import Node
from std_msgs.msg import String

class LangChainROSNode(Node):
    def __init__(self):
        super().__init__('langchain_ros_node')
        self.publisher = self.create_publisher(String, 'llm_responses', 10)

        # Initialize LangChain components
        self.llm = ChatOpenAI(model="gpt-4o", temperature=0.7)
        self.prompt = PromptTemplate.from_template(
            "Answer the following question based on the context: {question}"
        )
        self.chain = self.prompt | self.llm | StrOutputParser()

    def process_question(self, question):
        # Process question through LangChain
        response = self.chain.invoke({"question": question})

        # Publish response
        msg = String()
        msg.data = response
        self.publisher.publish(msg)

        return response
```

### LangChain with ROS 2 Actions

```python
# Example: LangChain with ROS 2 Actions
from langchain_core.tools import tool
from langchain.agents import create_tool_calling_agent
from langchain.agents import AgentExecutor

@tool
def move_robot(distance: float, direction: str):
    """Move the robot a certain distance in a specific direction"""
    # This would interface with ROS 2 action servers
    return f"Moving robot {distance} meters {direction}"

@tool
def detect_obstacles():
    """Detect obstacles in the robot's environment"""
    # This would interface with ROS 2 sensors
    return "Detected 2 obstacles at coordinates [1.5, 2.0] and [3.2, 1.8]"

class AIActionAgent(Node):
    def __init__(self):
        super().__init__('ai_action_agent')

        # Define tools
        tools = [move_robot, detect_obstacles]

        # Create LangChain agent
        llm = ChatOpenAI(model="gpt-4o")
        agent = create_tool_calling_agent(llm, tools, PromptTemplate.from_template(""))
        self.agent_executor = AgentExecutor(agent=agent, tools=tools)

    def process_instruction(self, instruction):
        # Process natural language instruction
        result = self.agent_executor.invoke({"input": instruction})
        return result
```

### Advanced GPT-4o with Context Management

```python
# Example: GPT-4o with conversation context
import openai
import rclpy
from rclpy.node import Node
from std_msgs.msg import String

class ContextAwareGPT(Node):
    def __init__(self):
        super().__init__('context_aware_gpt')
        self.publisher = self.create_publisher(String, 'context_responses', 10)

        # Initialize conversation context
        self.conversation_history = []

        # Initialize OpenAI client
        self.client = openai.OpenAI(api_key='your-openai-api-key')

    def process_with_context(self, user_input):
        # Add user message to conversation history
        self.conversation_history.append({"role": "user", "content": user_input})

        # Send to GPT-4o with context
        response = self.client.chat.completions.create(
            model="gpt-4o",
            messages=self.conversation_history,
            temperature=0.7
        )

        # Get response
        answer = response.choices[0].message.content

        # Add assistant response to history
        self.conversation_history.append({"role": "assistant", "content": answer})

        # Keep conversation history manageable
        if len(self.conversation_history) > 10:
            self.conversation_history = self.conversation_history[-10:]

        # Publish response
        msg = String()
        msg.data = answer
        self.publisher.publish(msg)

        return answer
```

```python
# Voice-controlled robot example
import speech_recognition as sr
import rclpy
from rclpy.node import Node
from std_msgs.msg import String

class VoiceControlledRobot(Node):
    def __init__(self):
        super().__init__('voice_controlled_robot')
        self.publisher = self.create_publisher(String, 'voice_commands', 10)

        # Initialize speech recognition
        self.recognizer = sr.Recognizer()

    def listen_and_process(self):
        with sr.Microphone() as source:
            print("Listening for voice command...")
            audio = self.recognizer.listen(source)

        try:
            # Convert speech to text
            text = self.recognizer.recognize_google(audio)

            # Publish voice command
            msg = String()
            msg.data = text
            self.publisher.publish(msg)

            return text

        except sr.UnknownValueError:
            return "Could not understand audio"
        except sr.RequestError as e:
            return f"Could not request results; {e}"
```

### 1. Error Handling and Fail-Safe Mechanisms

```python
# Example: Robust AI integration with error handling
import rclpy
from rclpy.node import Node
from std_msgs.msg import String

class RobustAIIntegration(Node):
    def __init__(self):
        super().__init__('robust_ai_integration')
        self.publisher = self.create_publisher(String, 'ai_responses', 10)

    def safe_ai_call(self, input_text):
        try:
            # Attempt AI call
            response = self.perform_ai_call(input_text)

        except Exception as e:
            # Handle AI call failure
            self.get_logger().error(f"AI call failed: {str(e)}")

            # Provide fallback response
            fallback_msg = String()
            fallback_msg.data = "AI service temporarily unavailable. Using default response."
            self.publisher.publish(fallback_msg)

            return "Fallback response"

    def perform_ai_call(self, input_text):
        # Actual AI call implementation
        pass
```

### 2. Rate Limiting and Resource Management

```python
# Example: Rate limiting for AI calls
import time
import rclpy
from rclpy.node import Node
from std_msgs.msg import String

class RateLimitedAI(Node):
    def __init__(self):
        super().__init__('rate_limited_ai')
        self.publisher = self.create_publisher(String, 'ai_responses', 10)
        self.last_ai_call_time = 0
        self.min_call_interval = 1.0  # Minimum 1 second between calls

    def ai_call_with_rate_limit(self, input_text):
        current_time = time.time()

        if current_time - self.last_ai_call_time < self.min_call_interval:
            # Too soon, return cached response or skip
            return "Rate limited"

        # Perform AI call
        response = self.perform_ai_call(input_text)
        self.last_ai_call_time = current_time

        return response
```

### 3. Logging and Monitoring

```python
# Example: Comprehensive logging for AI integration
import logging
import rclpy
from rclpy.node import Node
from std_msgs.msg import String

class MonitoredAI(Node):
    def __init__(self):
        super().__init__('monitored_ai')
        self.publisher = self.create_publisher(String, 'ai_responses', 10)

        # Setup logging
        self.logger = logging.getLogger('ai_integration')
        self.logger.setLevel(logging.INFO)

    def monitored_ai_call(self, input_text):
        # Log AI call start
        self.logger.info(f"Starting AI call with input: {input_text[:50]}...")

        start_time = time.time()

        try:
            # Perform AI call
            response = self.perform_ai_call(input_text)

            # Log success
            end_time = time.time()
            duration = end_time - start_time
            self.logger.info(f"AI call completed successfully in {duration:.2f}s")

            return response

        except Exception as e:
            # Log error
            self.logger.error(f"AI call failed after {time.time() - start_time:.2f}s: {str(e)}")
            raise
```
